A WordPress theme called Bro Barbershop.

== Description ==

Modern and eye-catching, this barbershop WordPress theme was made to present your hairdressing business in style. Create an outstanding portfolio for your services, maintain a high-concept blog, and make your business easily reachable online.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
